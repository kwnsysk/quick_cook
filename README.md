
# <Quick Cook>

## サイト概要
### サイトテーマ  
**簡単料理のレシピ**の情報を投稿し、共有することができるサイトです。  
扱うレシピは簡単に調理可能なレシピに限定し、
具材は**４種類までしか使用できない**という制限あり。  
**レシピの投稿は会員のみ**しか行えない。  
レシピの閲覧については**ログインをしなくとも可能**。  
会員数を増やす取り組みとして、気になるレシピのブックマーク機能を検討。  

### テーマを選んだ理由  
既存のレシピサイトは使用具材が多く、難しいレシピが多いと感じたため。  
所要時間や食材数(上限４個)、調理方法などの観点から簡単なレシピのみに限定する。  
Uber eatsなど簡単にご飯が手に入るツールがある中、  
料理をするハードルを下げることで、料理をする楽しさを提供したい。  

### ターゲットユーザー
* 料理をあまりしない人
* 一人暮らしの人
* 簡単に１品料理を増やしたい主婦

### 主な利用シーン
* 料理のレシピを考える時
* コンビニ・スーパーで買い物している時
* おかずを一品追加したい時

## 設計書
### 本サイトで扱うレシピに含まれる情報  
* 料理名
* 投稿者名
* 調理時間
* ジャンル（ご飯系/お菓子系）
* 調理手間情報（火を使わない・包丁を使わない　など　チェックボックス式）
* 具材１・具材２・具材３・具材４
* 調理方法（テキストボックス１つ）
